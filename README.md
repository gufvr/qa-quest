# QA Quest

Protótipo inicial de uma plataforma gamificada para aprendizado de Quality Assurance.

## Como abrir

Abra o arquivo `index.html` em qualquer navegador moderno. Não há dependências, instalação ou processo de build.

## Escopo atual

- identidade visual Tech Adventure;
- tela inicial responsiva;
- temas claro e escuro com preferência salva no navegador;
- apresentação da jornada e dos primeiros mundos;
- feedback visual ao acionar os botões de início;
- fundamentos de acessibilidade e suporte a movimento reduzido.
- primeira missão com cinco questões de múltipla escolha;
- feedback explicativo e pontuação de até 100 XP durante a sessão.

Persistência de progresso entre sessões ainda não foi implementada.
